## 배운것
모듈을 불러오는 4가지 방법을 공부하였다.  
1. 모듈 전체를 불러와 사용하는 방법
   - 불러오기 `import 모듈명`
   - 사용시 `모듈명.함수명()`
2. 모듈의 일부만 불러와 사용하는 방법
   - 불러오기 `from 모듈명 import 함수명`
   - 사용시 `함수명()`
3. 편리하게 사용할수 있도록 전부 불러오는 방법
   - 불러오기 `from 모듈명 import *`
   - 사용시 `함수명()`
   - 주의사항: 함수명이 겹치는 모듈 복수를 불러올때 주의 필요
4. 불러올때 별명을 지어주는 방법
   - 불러오기 `import 모둘명 as 별명`
   - 사용시 `별명.함수명()`
